# GoDat
Go items.dat Encoder/Decoder for Growtopia. Extremely fast btw.

# Features
- [X] ProtonHash Calculator.
- [X] Items.dat Encoder & Decoder.
- [X] Simple UI & User-Friendly.
- [X] Very Fast Processing.
